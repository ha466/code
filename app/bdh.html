Baby Dragon Hatchling

Sept/Oct 2025 | Scale-free Graph

A true post-transformer design using a living-graph style computation.

Overview & Core Idea

Not layered or looped in the classic sense. Uses a scale-free graph of locally interacting "neuron particles" where attention-like behavior emerges from dynamics.

Key Difference from Standard Transformers

Bio-inspired local message passing + Hebbian-like plasticity. True post-transformer living-graph design.

Memory System & Compute

Intrinsic memory and continuous adaptation achieved through continuous local interactions and structural plasticity.

Strengths & Real World Use Cases

Excellent reasoning (e.g., 97.4% on hard Sudoku Extreme). Strong native reasoning without heavy CoT. Continuous learning.

  • Custom agents on local hardware requiring continuous adaptation.
  • Long-horizon tasks needing structural plasticity.
