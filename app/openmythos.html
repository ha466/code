OpenMythos

2026 (kyegomez/OpenMythos + community analysis) | Strongly Recursive / Looped

A highly efficient recurrent-depth model reconstructing "Claude Mythos".

Overview & Core Idea

Prelude + single shared transformer block looped many times + Coda. Optional ACT halting. It leverages weight-tied recurrent depth and linear-time-invariant refinement across loops.

[ Architecture Diagram Placeholder ]

Key Difference from Standard Transformers

Weight-tied recurrent depth + linear-time-invariant refinement. Emergent subspace selectivity without massive layer stacking.

Memory System & Compute

Instead of deep static layers, it continuously passes latent representations through the same block, updating and refining memory iteratively across loops.

Strengths & Real World Use Cases

Extremely parameter-efficient "deep thinking". Adaptive compute matching problem difficulty. Suspected real-world frontier pattern.

  • Local runnable reasoning agents.
  • Custom iterative problem-solving bots.
