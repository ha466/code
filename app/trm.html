Tiny Recursive Model (TRM)

Oct 2025 (Samsung SAIT) | Pure Recursive / Tiny

A 7M parameter powerhouse that recursively refines its own answer.

Overview & Core Idea

Single tiny network (~2 layers, 7M params) that recursively refines its own answer and latent state with deep supervision.

[ Architecture Diagram Placeholder ]

Key Difference from Standard Transformers

Extreme parameter sharing + recursive refinement in one small net. Simpler and stronger than earlier hierarchical variants.

Memory System & Compute

Uses its own latent state to maintain context across many iterative steps, functionally acting as a deep recurrent memory without the parameter bloat.

Strengths & Real World Use Cases

Beats much larger LLMs on ARC-AGI, Sudoku, etc., with tiny data and params.

  • Tiny/local logical reasoning modules.
  • Embedded AI on highly constrained hardware.

Project Fit & Resources

Multiple PyTorch impls (lucidrains, Samsung repo, single-file versions).
