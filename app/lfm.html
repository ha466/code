Cantor Light Fractal Model (LFM)

2025–26 (Soulaiman Itani) | Hybrid fractal

A fractal hierarchical transformer processing input at multiple scales concurrently.

Overview & Core Idea

Maintains parallel embedding layers for tokens, words, phrases, sentences, and documents. Lower-level blocks are pooled to higher levels, while higher levels provide top-down conditioning.

[ Architecture Diagram Placeholder ]

Key Innovation vs Standard Transformers

Structure is multi-resolution instead of sequential depth. Uses fractal projections (upward summarization and downward conditioning). Introduces a specialized memory anticipation module.

Memory System & Compute

Hierarchical memory anticipating associations across local to global contexts.

Strengths

Linear compute scaling ~O(n) instead of quadratic for long contexts. Early reports show 40–60% faster training convergence. Naturally integrates global and local reasoning.

Weaknesses

Very early-stage (mostly a conceptual design). Complex multi-level state and gradients. Not yet validated on many tasks.

Resources & Open Source

Described in an IEEE Computer article by Itani; no public code as of 2026.
