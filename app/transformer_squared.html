Transformer² (Transformer Squared)

Jan 2025 (Sakana AI) | Self-adapting

A model that dynamically literally changes its brain per prompt.

Overview & Core Idea

Self-adapts its own weights at inference time instead of fixed recursion or static layers. Uses a two-pass system: analyzes task → dynamically mixes task-specific "expert" vectors.

[ Architecture Diagram Placeholder ]

Key Difference from Standard Transformers

Real test-time self-adaptation via SVD + RL-trained adapters. No full fine-tuning or external loops required.

Memory System & Compute

Task-specific adaptations are dynamically mixed into singular components of weight matrices, altering the core associative memory for the current context.

Strengths & Real World Use Cases

Handles unseen tasks dynamically. Extremely efficient adaptation on the fly.

  • Personalized or multi-domain local agents.
  • Dynamic zero-shot task switching.
