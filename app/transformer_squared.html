Transformer²

2025 (Sakana AI) | Hybrid (attention + dynamic parameter adaptation)

A framework for self-adaptive Transformers.

Overview & Core Idea

Instead of fixed weights, the model learns to adapt its weights on the fly for each task. Uses a two-pass mechanism: a dispatch network identifies task properties, then task-specific expert vectors are injected.

[ Architecture Diagram Placeholder ]

Key Innovation vs Standard Transformers

Allows the model’s “brain” to change per prompt without retraining. Only selected components of weight matrices are modified at inference time, driven by RL-trained vectors.

Memory System & Compute

Dynamic weight adaptation acts as a task-specific associative memory switch.

Strengths

More efficient than full fine-tuning. Superior performance across tasks and modalities. The model essentially configures itself.

Weaknesses

Adds complexity (requires training adaptation module and inference routing). Extra inference overhead to combine expert vectors. Newer concept with fewer benchmarks.

Resources & Open Source

ICLR 2025 paper; open-source code on GitHub (SakanaAI/self-adaptive-llms).
